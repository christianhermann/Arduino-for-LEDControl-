# Enhanced LED Control Arduino

An Arduino-based LED control system that provides enhanced functionality for controlling LED brightness and states through serial communication.

## Features

- Serial communication interface for LED control
- Adjustable LED brightness using PWM (Pulse Width Modulation)
- Real-time control through Serial Monitor or external applications
- Complex on/off protocols
